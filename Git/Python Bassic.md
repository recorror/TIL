**변수**
- 변수란 ? 
  - 데이터를 저장하기 위해서 사용
  - 변수를 사용하면 복잡한 값들을 쉽게 사용할 수 있음(추상화)
- 동일 변수에 다른 데이터를 언제든 할당(저장)할 수 있기 때문에, '변수'라고 불림.
- 식벽자의 이름은 연문 알파벳, 언더스코어(_), 숫자로 구성.
- 첫 글자에 숫자가 올 수 없음
- 길이 제한이 없고, 대소문자를 구변
- *what is 예약어 ???*
> 예약어란 먼저 예약이 된 언어라는 뜻으로 기본적으로 사용할 수 있는 단어, 명령어를 뜻한다. 예약어는 클래스, 함수, 변수 등의 이름으로 사용 할 수 없다.
